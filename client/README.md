# 🚀 Team Task Manager

A full-stack web application for managing team projects and tasks with role-based access.

## 🔥 Features
- User Authentication (Admin & Member)
- Project & Team Management
- Multi-user Task Assignment
- Dashboard with analytics
- Task status tracking (Todo, In Progress, Done)
- Overdue task detection

## 🛠 Tech Stack
- Frontend: React.js, Tailwind CSS
- Backend: Node.js, Express.js
- Database: MongoDB
- Auth: JWT

## 🚀 How to Run
### Backend
cd server
npm install
npm run dev

### Frontend
cd client
npm install
npm start
