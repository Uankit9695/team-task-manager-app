# 🚀 Team Task Manager (Full-Stack MERN App)

A complete team task management web application where users can create projects, assign tasks, and track progress with role-based access control.

## 📌 Features

### 🔐 Authentication
- User Signup & Login
- JWT-based authentication
- Role-based access (Admin / Member)

### 👥 Project & Team Management
- Create projects
- Add multiple team members
- Assign members dynamically

### ✅ Task Management
- Create tasks
- Assign tasks to multiple users
- Update task status (Todo / In Progress / Done)
- Due date tracking

### 📊 Dashboard
- Total tasks
- Completed tasks
- Pending tasks
- Overdue tasks
- Task distribution chart

## 🛠 Tech Stack
